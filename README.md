# system-admin-articles
Wiki for documenting installation, configuration, maintenance, and using of core IT and cybersecurity softwares and services. Includes Active Directory, Microsoft 365, various Windows Server roles, Wazuh SIEM, and more!

trixie.local is my virtualized sandbox network for testing, deploying, and documenting setup of network security software and important IT services. including active directory domain controller, wazuh siem, ansible for automation, kubernetes for clustering and containerization, digitalocean for cloud computing, all on a proxmox baremetal server.
